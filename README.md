# CTF Write-Ups 🛡️

A curated collection of my **Capture The Flag (CTF)** write-ups, solutions, and technical notes.

This repository documents my approach to solving challenges across multiple cybersecurity domains, with a focus on **clear methodology**, **reproducible steps**, and **proper tooling**.

---

## 📌 Contents

Write-ups include challenges from categories such as:

- 🔐 Cryptography  
- 🌐 Web Exploitation  
- 🧠 Secure Coding / Code Review  
- 🐧 Linux Privilege Escalation  
- 🧾 Digital Forensics & Steganography  
- 📡 Network Forensics (PCAP analysis, DNS/HTTP exfil)  
- 📱 Mobile Security / APK Static Analysis  

---

## 🧰 Tools & Techniques Used

Some tools commonly used across the write-ups:

- Wireshark
- Burp Suite
- CyberChef
- Nmap
- exiftool / binwalk
- strings / grep / regex analysis
- jadx / apktool
- Python scripting for decoding and automation

## 📁 Repository Structure

Each folder typically contains:
- `writeup.md` / `README.md`
- screenshots (`screenshots/`)
- scripts used during solving (`.py`, `.sh`, etc.)
- extracted challenge artifacts (when allowed)
